TRACE-AML

A local-first investigative workspace for live recognition, entity tracking, incident review, and evidence intelligence. This desktop shell packages the operational workflow into a portable forensic interface without moving the core system away from its original architecture.

Designed and developed by Utkarsh Chandra. The desktop flow, interface, and source architecture remain grounded in the original TRACE-AML repository and its underlying local service model.
